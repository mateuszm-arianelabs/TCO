# TCO

## Run Localy

required wallet with:
- Minimal value of BNB tokens just to estimate gas and call allowance
- 0.1 USDC on BNB Chain for estimate gas

1. Clone repo
2. `pnpm install`
3. Create `.env` and pass PRIVATE_KEY
4. `pnpm dev`
